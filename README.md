# 📝 To-Do List App

A simple command-line To-Do List application built with Python.  
This app allows users to manage daily tasks by adding, viewing, and removing them.

---

## 🚀 Features

- View all tasks
- Add new tasks
- Remove tasks
- Simple and easy-to-use menu
- Runs in the terminal (CLI)

---

## 🧠 How it works

The application runs in a loop and displays a menu with options:

1. View Tasks  
2. Add Task  
3. Remove Task  
4. Exit  

Users select an option by entering a number, and the app performs the action.
